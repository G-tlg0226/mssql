## 安装
> 1. git clone https://git.oschina.net/ycg520520/nodeserv_api.git
> 2. cd nodeserv_api
> 3. npm install #安装项目运行的npm包

## 部署
> 1. npm install -g pm2 # 全局安装pm2服务器管理工具
> 2. npm start 或者 npm run dev # 启动开发服务
> 3. npm run pro # 启动服务
