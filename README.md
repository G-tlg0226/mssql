## node-sqlo  
> 将nodejs操作SQL Server简单化

## Usage
> 需要nodejs 7 或 更高
``` javascript
    npm install sqlo 
```

## Documentation
 - creatDBC 创建数据库连接
 - closeDBC 关闭数据库连接
 - creatDBP 创建数据库连接池
 - closeDBP 关闭数据库连接池
 - displayDBPI 返回指定连接池的信息
 - insert 新增
 - delete 删除
 - update 修改
 - select 查询
 - use 数据库(数据库服务器)切换，
 - query 执行sql
 - batch 批量执行sql
 - transaction 事务(开发中)
 - orm（提供SQL Server的orm支持，未来会支持）
